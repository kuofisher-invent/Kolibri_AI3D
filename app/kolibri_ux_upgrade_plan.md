# 《Kolibri Ai3D UX 升級計畫（SketchUp 體感 + AI 輔助）》v1.0

---

## 🎯 目標

將 Kolibri Ai3D 從「功能完整的 3D 工具」升級為：

👉 **具備 SketchUp 操作體感 + AI 輔助能力的建模系統**

---

## 🧠 核心理念

### ❌ 錯誤方向
- AI 直接建模
- UI 很漂亮但操作卡

### ✅ 正確方向
- 人主導操作（滑鼠體感）
- AI 輔助決策
- 即時回饋（數值 / 推論 / 吸附）

---

## 🚀 第一階段：體感強化（最優先）

### 1️⃣ 游標旁數值顯示（必做）

#### 功能
- Move / PushPull / Draw 時
- 在游標旁顯示距離

#### UI 範例
```
↔ 1250 mm
↑ 3000 mm
```

#### 效果
👉 使用者感覺「在拉物件」而不是「輸入數值」

---

### 2️⃣ 任意工具皆可選取（關鍵）

#### 現況
- 必須切換 Select 工具

#### 改善
```
任何工具下：
→ 點物件 = 選取
→ 拖曳 = 操作
```

#### 效果
👉 操作流不中斷（SketchUp 核心體驗）

---

### 3️⃣ Push/Pull 體驗補完

#### 補強功能
- 雙擊 = 重複距離
- 游標顯示拉伸距離
- 虛線參考邊

#### 效果
👉 變成真正「雕塑」體驗

---

## 🧲 第二階段：Inference 強化

### 1️⃣ 軸向鎖定提示
```
On Red Axis
On Green Axis
On Blue Axis
```

### 2️⃣ 推論 + 數值整合
```
Midpoint — 2500 mm
```

### 3️⃣ 面吸附（進階）
- 支援非地面繪圖

---

## 🎮 第三階段：操作回饋

### 1️⃣ Hover 高亮（Edge / Face）

#### 現況
- 只有 object

#### 改善
- edge highlight
- face highlight

---

### 2️⃣ 游標圖示升級

| 工具 | 游標 |
|------|------|
| Move | 十字 |
| Push | 上下箭頭 |
| Line | 鉛筆 |

---

### 3️⃣ 面正反顯示

| 面向 | 顏色 |
|------|------|
| 正面 | 白 |
| 背面 | 藍 |

---

## 🤖 第四階段：AI 輔助層

### ❌ 不建議
- AI 直接 create_box

### ✅ 建議

AI 負責：
- 推薦工具
- 補數值
- 推論修正
- 操作建議

---

### UI 設計

```
🤖 建議：
→ 使用 Push/Pull 拉高 2800 mm
→ 偵測到牆體輪廓
```

---

## 🧩 第五階段：Hybrid 工作流

### 自動（AI）
- CAD → 語意辨識
- 初步建模（80%）

### 手動（使用者）
- 修正
- 細節建模
- 結構調整
