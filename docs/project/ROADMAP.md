# Kolibri_Ai3D 功能清單與待辦追蹤

> 更新日期：2026-03-26

---

## 已完成功能

### 視覺渲染
天空漸層背景、地面格線、XYZ 軸線、方向光+環境光、邊線描繪、深度測試、Alpha 混合、平面投影陰影、面正反面不同色、平滑法線、6 種顯示模式（著色/線框/X光/隱藏線/單色/草稿）、程序化材質紋理

### 相機
環繞/平移/縮放、全部顯示、WASD 行走、透視/正交切換、標準視圖（前/後/左/右/上/等角）、場景視角儲存

### 繪圖工具
線段/弧線/矩形/圓形、方塊/圓柱/球體（SketchUp 風格互動建立）、推拉（面級別六面）、移動/旋轉/縮放、偏移/跟隨/量測/油漆桶/標註/文字、橡皮擦/群組/元件

### 推斷/捕捉
端點/中點/原點/面中心捕捉、軸向 RGB 引導線、面上繪圖、格線捕捉、Shift 鎖軸、Inference 2.0 評分管線

### 編輯操作
Undo/Redo（50 步）、Ctrl+Move 複製、陣列複製 "3x"、雙擊推拉重複、框選/多選/全選、精確尺寸輸入

### 材質系統
29 種預設材質（7 大類）、自訂 RGBA、PBR 粗糙度/金屬感、材質預覽球、油漆桶工具

### 組織管理
群組（雙擊隔離編輯）、元件實例（改一個全改）、圖層/標籤（可見性切換）、場景大綱

### 檔案 I/O
.k3d 存取、OBJ/STL/GLTF/DXF 匯出入、PNG/JPG 截圖、自動儲存、最近檔案、拖放開啟

### CAD 匯入
DXF 語意解析（LINE/POLYLINE/ARC/CIRCLE/TEXT/DIMENSION/INSERT）、DWG 二進制掃描、PDF 向量路徑提取、座標自動歸零、語意偵測（柱/梁/板/軸線）、使用者確認面板

### 鋼構模式
Grid/Column/Beam/Brace/Plate/Connection 6 工具、H 型鋼 3 件式建模、AABB 碰撞偵測、群組選取

### AI 整合
MCP Server 內建（11 工具）、AI 審計日誌、測試入口

### 架構
三層 Store 分離（SceneStore/ViewerState/EditorState）、扁平節點字典（parent_id 層級）、CLAUDE.md 規範
