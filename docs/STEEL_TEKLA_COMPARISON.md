## 4. Kolibri 差異化優勢

1. **輕量化** — 50MB vs 10GB，2 秒啟動
2. **AI 原生** — MCP Server 讓 AI 直接操控建模
3. **CNS 台灣標準** — 原生 CNS 386/2473/4435
4. **AISC 內建** — 接頭計算含 360-22 驗算
5. **3D+2D+AI 統一** — 建模→接頭→報表→施工圖→NC/IFC 全在一個 APP
6. **零依賴** — 不需授權伺服器、不需網路
7. **跨平台** — Rust/wgpu，可編譯至 Windows/Mac/Linux
